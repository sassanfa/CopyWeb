# CopyWeb v1.3.0

نسخه‌ی 1.3.0 یک ارتقای بزرگ برای آرشیو، بررسی و انتشار نسخه‌ی آفلاین سایت است.

## مهم‌ترین تغییرات

- پیش‌نمایش داخلی پروژه روی `localhost` و گزارش لینک‌های خراب محلی.
- حالت `Watch` با hash manifest؛ فقط صفحات تغییرکرده دوباره دانلود می‌شوند.
- ذخیره‌ی اشتراکی تصاویر در `%LOCALAPPDATA%\CopyWeb\SharedAssets` و حذف دانلود تکراری.
- اسکرین‌شات رندرشده‌ی خودکار قبل و بعد از آرشیو در پوشه‌ی `screenshots`.
- اعلان پایان دانلود با صدای سیستم، Windows notification، Webhook و ایمیل.
- داشبورد پروژه با تعداد صفحات، خطاها و حجم فایل‌ها بر اساس نوع.
- چرخش خودکار پروفایل‌های پروکسی سالم بر اساس سلامت و زمان پاسخ.
- انتشار به ZIP، پوشه‌ی آماده‌ی IIS، FTP و SFTP با OpenSSH/SSH Key.
- CLI نسخه‌ی 1.3.0 با پشتیبانی از `--resume`، proxy، retry، timeout و محدودیت سرعت.
- ورود کاربر از داخل WebView2 و استفاده از نشست رمزنگاری‌شده برای دانلود صفحات خصوصی.

## اصلاحات فنی

- checkpoint و `assets-manifest.json` به‌صورت اتمیک ذخیره می‌شوند.
- تصویرهای تکراری با URL، query string یا محتوای یکسان دوباره دانلود نمی‌شوند.
- وضعیت هر لینک، درصد فایل جاری، سرعت، صف، خطا و فضای آزاد نمایش داده می‌شود.
- تنظیمات اعلان، تم، زبان و پروکسی در فایل تنظیمات برنامه ذخیره می‌شوند.

## فایل‌های انتشار

- `CopyWeb-Setup-1.3.0.exe` — نصب‌کننده‌ی Windows
- `CopyWeb-Portable-1.3.0.zip` — نسخه‌ی Portable
- `CopyWeb-Portable.exe` — اجرای مستقیم بدون نصب
- `CopyWeb.exe` — فایل اجرایی اصلی
- `CLI.md` — راهنمای خط فرمان

## نکته‌ی ارتقا

برای اجرای SPA و گرفتن screenshot به WebView2 Runtime نیاز است. پروژه‌های قدیمی بدون manifest منابع، هنگام اجرای بعدی به‌صورت خودکار manifest جدید می‌سازند.
