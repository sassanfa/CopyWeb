# CopyWeb v1.2.0

## فایل‌های انتشار

- `CopyWeb-Setup-1.2.0.exe`: نصب‌کننده‌ی کامل Windows
- `CopyWeb.exe`: نسخه‌ی اجرایی Portable
- `CopyWeb-Portable-1.2.0.zip`: بسته‌ی Portable
- `CLI.md`: راهنمای اجرای CLI و Headless
- `INSTALL.md`: راهنمای نصب و ساخت Setup
- `CopyWeb.iss`: اسکریپت Inno Setup

## CLI

```powershell
CopyWeb.exe --cli --url https://example.com --output C:\Sites\example
CopyWeb.exe --cli --help
CopyWeb.exe --cli --self-test
```

قابلیت CLI داخل همان `CopyWeb.exe` قرار دارد و فایل جداگانه‌ای لازم نیست.
