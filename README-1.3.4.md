# CopyWeb 1.3.4

CopyWeb یک ابزار Windows برای دریافت نسخه‌ی آفلاین سایت، ذخیره‌ی منابع و ساخت آرشیو قابل بررسی است.

## امکانات شاخص

- دانلود صفحات، تصاویر، CSS، JavaScript، فونت، WebP و AVIF
- Resume، checkpoint اتمیک، Retry و حذف موارد انتخاب‌شده
- جلوگیری از دانلود تکراری با URL نرمال‌شده و Hash
- پروکسی HTTP/HTTPS/SOCKS5 با نگهداری امن اطلاعات با Windows DPAPI
- حالت «ذخیره زنده» و «کپی وبی» با نمایش وضعیت منابع و Highlight سبز
- بازکردن لینک‌های `target=_blank` داخل همان مرورگر برای ثبت کامل صفحات
- بازنویسی مسیرهای HTML/CSS برای اجرای صحیح نسخه‌ی آفلاین
- گزارش‌ها، فیلتر وضعیت، خروجی TXT/CSV/JSON و Crash Log
- پیش‌نمایش آفلاین، اعتبارسنجی آرشیو، Snapshot و Visual Diff
- CLI برای اجرای بدون رابط گرافیکی و self-test
- داشبورد تیره‌ی جدید با عملیات سریع و دکمه‌های واضح «تنظیمات پیشرفته» و «ذخیره زنده»
- نمایش پروژه‌های واقعی اخیر به‌جای اطلاعات نمایشی، همراه با تاریخ، تعداد فایل، حجم، درصد و خطا
- نمودار زنده‌ی پیشرفت، درصد کل، درصد فایل جاری، سرعت، ETA و فایل در حال دانلود
- خلاصه‌ی تصویری وضعیت‌های تکمیل‌شده، در حال دانلود، در صف و ناموفق
- بارگذاری پروژه با کلیک روی ردیف آن و به‌روزرسانی خودکار داشبورد بعد از پایان یا توقف
- رابط واکنش‌گرا، دکمه‌های یکدست، آیکون‌های هماهنگ و کنتراست کامل در حالت ساده و پیشرفته

## اجرا

برای استفاده‌ی معمولی `CopyWeb.exe` را اجرا کنید. برای نسخه‌ی قابل حمل از `CopyWeb-Portable.exe` یا فایل ZIP استفاده کنید.

```text
CopyWeb.exe --cli --help
CopyWeb.exe --cli --self-test
```

برای حالت‌های SPA، CAPTCHA، ورود کاربر و ذخیره‌ی زنده، Microsoft WebView2 Runtime باید روی Windows نصب باشد.

## حالت ساده و پیشرفته

- «شروع دانلود جدید» فرم ساده را باز می‌کند و تنظیمات لازم را خودکار انتخاب می‌کند.
- «تنظیمات پیشرفته» کنترل عمق، حداکثر صفحه، Sitemap، Canonical، Proxy، Retry، Timeout، سرعت و محل ذخیره را نمایش می‌دهد.
- از نوار کناری یا دکمه‌ی «شروع» می‌توانید هر زمان به داشبورد برگردید.
