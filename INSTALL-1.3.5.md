# نصب CopyWeb 1.3.5

## روش پیشنهادی

فایل `CopyWeb-Setup-1.3.5.exe` را اجرا کنید و مراحل نصب را ادامه دهید. میانبر برنامه در Start Menu و Desktop ساخته می‌شود.

## نسخه Portable

فایل `CopyWeb-Portable-1.3.5.zip` را Extract کنید و `CopyWeb-Portable.exe` را اجرا کنید. این نسخه به نصب .NET نیاز ندارد.

## نسخه معمولی

فایل `CopyWeb-Windows-x64-1.3.5.zip` را Extract کنید و `CopyWeb.exe` را اجرا کنید.

## پیش‌نیاز مرورگر داخلی

برای قابلیت‌های ورود، SPA، ذخیره زنده و کپی وبی، Microsoft Edge WebView2 Runtime باید روی Windows نصب باشد.

## پیش‌نمایش آفلاین

پیش‌نمایش روی localhost اجرا می‌شود. اطلاعات ورود محلی:

```text
Username: admin
Password: admin
```

این حساب به حساب اصلی وب‌سایت ارتباطی ندارد.
