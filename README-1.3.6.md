# CopyWeb 1.3.6

CopyWeb یک برنامهٔ Windows برای بررسی، دانلود، ادامه‌دادن و پیش‌نمایش آفلاین سایت‌ها است. نسخهٔ 1.3.6 روی شروع بدون پرش، دریافت کامل‌تر WebP/AVIF و مدیریت پروژه‌های ذخیرهٔ زنده تمرکز دارد.

## قابلیت‌های برجسته

- دانلود سریع با یک کلیک یا تنظیمات پیشرفته
- Splash متحرک شفاف، شروع بدون flicker و نوار عنوان Active/Inactive
- توقف، Resume و بازیابی checkpoint
- ذخیرهٔ HTML، CSS، JavaScript، تصاویر، فونت و رسانه
- ذخیرهٔ زنده با مرورگر داخلی، Cookie و نشست کاربر
- مسیر امن WebView2 در LocalAppData برای جلوگیری از خطای Access Denied
- تشخیص WebP/AVIF از URL، MIME و محتوای واقعی فایل
- جلوگیری از دانلود تکراری تصویر با SHA-256
- پروژه‌های زنده با نشان «● زنده» و Live Server روی localhost
- حذف امن پروژه از داشبورد با انتقال به Recycle Bin
- CLI، Portable، نسخهٔ معمولی و Setup ویندوز
- رابط فارسی/انگلیسی، تم تیره و گزارش‌های قابل خروجی

## ورود آفلاین

در آرشیوهای خصوصی آماده‌شده برای پیش‌نمایش محلی، نام کاربری و رمز پیش‌فرض:

`admin / admin`

## اجرای نسخه‌ها

- نسخهٔ معمولی: فایل ZIP را Extract و `CopyWeb.exe` را اجرا کنید.
- Portable: فایل ZIP را Extract و `CopyWeb-Portable.exe` را اجرا کنید.
- Setup: فایل `CopyWeb-Setup-1.3.6.exe` را اجرا کنید.
- CLI: از `CopyWeb-CLI.exe --help` استفاده کنید.
