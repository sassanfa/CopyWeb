# CopyWeb 1.3.5

CopyWeb یک ابزار Windows برای دریافت نسخهٔ آفلاین وب‌سایت‌ها، ذخیرهٔ صفحه‌ها و منابع آن‌ها و مرور آرشیو روی localhost است.

## تغییرات شاخص این نسخه

- هماهنگ‌سازی رنگ، کنترل‌ها و خوانایی تمام پنجره‌های داخلی با رابط اصلی
- بازطراحی ظاهر «ذخیره زنده» و فهرست لینک‌ها در تم تیره
- اجرای پایدار پیش‌نمایش آرشیو روی localhost
- ورود محلی پیش‌نمایش با نام کاربری `admin` و رمز `admin`
- دکمهٔ خروج از حساب محلی و نشست HttpOnly
- حذف CAPTCHA فقط از نسخهٔ آفلاین؛ روند ورود و حل CAPTCHA سایت اصلی بدون تغییر است
- آزمون خودکار ورود محلی، رمز اشتباه، نشست، بازپخش آرشیو و حذف CAPTCHA آفلاین
- به‌روزرسانی نسخهٔ رابط، About، CLI، Update Checker، Portable و Setup به 1.3.5

## استفاده از سایت‌های نیازمند ورود

1. «ورود به» یا «ذخیره زنده» را باز کنید.
2. در سایت اصلی وارد شوید و صفحه‌هایی را که باید ذخیره شوند باز کنید.
3. برای مشاهدهٔ آرشیو از پیش‌نمایش آفلاین استفاده کنید.
4. در صفحهٔ ورود محلی، `admin` را برای نام کاربری و رمز وارد کنید.

حساب `admin/admin` تنها برای localhost است. اطلاعات حساب واقعی سایت در آرشیو ذخیره نمی‌شود و عملیات وابسته به سرور اصلی در حالت آفلاین قابل اجرا نیست.

## اجرای CLI

```text
CopyWeb.exe --cli --help
CopyWeb.exe --cli --self-test
CopyWeb.exe --cli --url https://example.com --output C:\Sites\example
```
